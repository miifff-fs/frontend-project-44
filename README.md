# Brain Games

Набор из пяти консольных математических игр.

## Игры:
1. **brain-even** — проверка числа на чётность
2. **brain-calc** — вычисление арифметического выражения
3. **brain-gcd** — поиск наибольшего общего делителя
4. **brain-progression** — поиск пропущенного числа в прогрессии
5. **brain-prime** — проверка числа на простоту

## Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/miifff-fs/frontend-project-44.git
cd frontend-project-44

# Установите зависимости
npm install

# Установите пакет глобально
npm link
```

## Запуск игр
```bash
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
